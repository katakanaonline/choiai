# 監視通知サービス設計書

## サービス概要

**サービス名**: ちょいウォッチ（仮）
**コンセプト**: 「気になるアレ、自動で監視して即通知」

既存の freelance_checker.py のアーキテクチャを流用し、
汎用的な監視・通知サービスとしてパッケージ化。

---

## 技術スタック（既存資産）

| 技術 | 用途 | 流用元 |
|------|------|--------|
| **Playwright** | ブラウザ自動化・スクレイピング | freelance_checker.py |
| **Gmail API** | メール監視 | freelance_checker.py |
| **ntfy.sh** | プッシュ通知（無料） | freelance_checker.py |
| **LINE Notify** | LINE通知 | 新規追加可 |
| **Slack Webhook** | Slack通知 | 新規追加可 |
| **cron-job.org** | 定期実行（無料） | 全プロジェクト |
| **Supabase** | データ保存・履歴管理 | ai-researcher |

---

## サービスメニュー

### プランA: シンプル監視（スポット）
**価格**: 5,000〜15,000円

- 対象サイト1つ
- 監視項目1〜3つ
- 通知先1つ（LINE/Slack/メール）
- 納品: Python スクリプト + 設定手順書

**想定案件例**:
- 「特定商品の価格変動を監視したい」
- 「競合サイトの更新を知りたい」
- 「在庫復活を即通知してほしい」

---

### プランB: 定期監視セット（月額）
**価格**: 10,000〜30,000円/月

- 対象サイト3つまで
- 監視項目5つまで
- 通知先複数対応
- 週次レポート（PDF）
- 障害対応・メンテナンス込み

**想定案件例**:
- 「EC競合の価格監視を継続的に」
- 「複数求人サイトの新着監視」
- 「口コミ・レビューの新着監視」

---

### プランC: カスタム開発（スポット）
**価格**: 30,000〜100,000円

- フルカスタム開発
- ダッシュボード構築
- API連携
- 複雑な条件判定

**想定案件例**:
- 「社内システムと連携した監視ツール」
- 「複数ECモールの在庫一元監視」
- 「AI分析付きの競合監視システム」

## アーキテクチャ

```
┌─────────────────────────────────────────────────────────┐
│  定期実行 (cron-job.org / Windows Task Scheduler)       │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  監視スクリプト (Python + Playwright)                    │
│  - config.json で監視設定を管理                         │
│  - 前回値との差分検出                                   │
│  - 条件判定（閾値、キーワード等）                       │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  データ保存 (Supabase / SQLite)                         │
│  - 監視履歴                                             │
│  - 変更ログ                                             │
│  - 通知履歴                                             │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  通知配信                                               │
│  - ntfy.sh (スマホプッシュ)                             │
│  - LINE Notify                                          │
│  - Slack/Discord Webhook                                │
│  - メール (SendGrid)                                    │
└─────────────────────────────────────────────────────────┘
```
