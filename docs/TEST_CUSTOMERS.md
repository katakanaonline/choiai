# ちょいマーケ テスト顧客候補リスト

## 目的

Phase 1 MVPのテスト導入先を見つける。
月額1〜2万円で価値を感じてもらえる顧客を探す。

---

## ターゲット条件

### 理想的な顧客像

| 条件 | 詳細 |
|------|------|
| 業種 | 地域密着型サービス業（来店型） |
| 規模 | 1〜10店舗 |
| 現状 | GBP登録済みだが放置気味 |
| 課題 | 口コミ対応が遅い、順位を把握していない |
| 予算 | 月2〜5万円のマーケ予算あり |
| ITリテラシー | 低〜中（ダッシュボードで完結する必要あり） |

### 優先業種

1. **飲食店**（ラーメン、居酒屋、カフェ）
   - 口コミが購買に直結
   - 競合が多く順位変動が激しい

2. **美容院・サロン**
   - 新規顧客獲得にGBP重要
   - 口コミ返信を重視

3. **クリニック**（歯科、整骨院、皮膚科）
   - 信頼性が重要
   - 低評価口コミへの対応ニーズ

4. **不動産仲介**
   - 地域検索でのMEO重要
   - 競合との差別化ニーズ

## 候補リスト

### A. クラウドワークス案件経由

クラウドワークスで応募したMEO案件から、逆提案できる可能性あり。

| 案件 | 業種 | 備考 |
|------|------|------|
| GBP投稿代行（継続案件） | 不明 | 投稿代行ニーズあり → 監視もセット提案 |
| GBP編集作業 | 複数店舗 | 編集代行 → 監視で価値追加 |
| MEO対策（都内パン屋） | パン屋 | 小規模店舗、監視ニーズあり |
| GBP設定代行（歯科7院） | 歯科 | 複数店舗、まとめて提案可能 |

**アプローチ**:
1. 案件受注後に信頼関係構築
2. 「監視ツールも使ってみませんか？」と追加提案
3. 初月無料 or 半額でテスト導入

### B. 知人・紹介経由

| カテゴリ | 候補 | アプローチ |
|----------|------|-----------|
| 知人経営 | - | LinkedIn、Facebook等で飲食店経営者を探す |
| 前職関係 | - | 元同僚で独立した人 |
| 地元 | - | 実家近くの店舗 |

### C. 直接アプローチ

| エリア | 業種 | 方法 |
|--------|------|------|
| 新宿 | ラーメン店 | 口コミ返信していない店舗にDM |
| 渋谷 | 美容院 | 評価が下がっている店舗にDM |
| 池袋 | 居酒屋 | GBP投稿が止まっている店舗にDM |

**DMテンプレート**:

```
件名: Googleマップの口コミ管理でお困りではないですか？

〇〇様

突然のご連絡失礼いたします。
マーケティングツール「ちょいマーケ」を開発しております石田と申します。

Googleマップで〇〇様の店舗を拝見し、
口コミ対応にお時間を取られているのではないかと思いご連絡いたしました。

当サービスでは、
・新着口コミの即時通知
・AIによる返信文の自動生成
・検索順位の毎日モニタリング

を月額2万円でご提供しております。

もしご興味がございましたら、
1ヶ月間の無料トライアルをご案内させていただきます。

よろしければ一度お話させていただけませんでしょうか。
```

## テスト顧客の条件

### 必須条件

- [ ] Googleビジネスプロフィール登録済み
- [ ] 口コミが10件以上ある
- [ ] 連絡先（メール or 電話）が判明

### 優先条件

- [ ] 過去3ヶ月の口コミに返信していない
- [ ] 競合が多いエリア・業種
- [ ] ITツールに抵抗がなさそう（SNS運用している等）

## リスク・注意点

1. **無料に慣れさせすぎない**
   - 無料期間は1ヶ月まで
   - 価値を感じたら有料化を明確に

2. **サポート工数**
   - テスト段階は手厚くなりがち
   - 自動化・セルフサービスを意識

3. **契約書**
   - テスト段階でも簡易契約書を用意
   - 個人情報の取り扱いを明記
